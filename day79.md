## ☀️ Morning Block (2 hours): Profile Sections

### Headline Formula

```
❌ "Student at [University]"
❌ "Intern at [Company]"  
❌ "Looking for opportunities"

✅ "Aspiring Data Engineer | SQL, Python, AWS, dbt, Airflow | Building Data Pipelines"
✅ "Data Engineer | SQL · Python · AWS · dbt · Airflow | 5 Portfolio Projects on GitHub"
✅ "Junior Data Engineer | ETL Pipelines · Data Modeling · AWS Cloud | Open to roles in Singapore"
```

The pipe character `|` or middle dot `·` looks clean. Include:
- What you are (Data Engineer)
- Top 4-5 skills (keywords for search)
- Differentiator or call to action

### About Section (200-300 words)

```
I'm an aspiring Data Engineer building end-to-end data solutions with SQL, Python, and AWS.

Over the past 3 months, I've completed an intensive self-directed data engineering program covering:

🔧 Technical Skills:
- SQL: Complex joins, CTEs, window functions, query optimization
- Python: ETL scripting, Pandas, API integration, data quality
- Cloud (AWS): S3, Glue, Athena, RDS, IAM — serverless data pipelines
- Data Tools: dbt, Airflow, PySpark, Docker, GitHub Actions

📊 What I've Built:
1. Food Delivery Data Warehouse — star schema + SCD Type 2 + Airflow orchestration

4. dbt Analytics Project — staging → marts with 36+ automated tests

🎯 What I'm Looking For:

📬 How to Reach Me:
- GitHub: github.com/yourname (all projects with full documentation)
```
