## 🎯 Why This Plan Works for Fresh Grad SG/MY

**What employers actually test at fresh grad level:**
1. SQL — write complex queries live (non-negotiable)
2. Python — write ETL scripts (non-negotiable)
3. Data modeling — "design a schema for X"
4. ETL pipeline understanding — "how would you build pipeline for X?"
5. Communication + documentation

**What's overkill for fresh grad (not spending full weeks on):**
- Spark deep-dive → enough to know what it is, Glue covers this
- Kafka/Streaming → senior-level, awareness is enough
- Terraform/IaC → DevOps territory
- Cloud certification → portfolio matters more than cert
- Docker mastery → 1 day of basics is enough

**This roadmap focuses on:**
- ✅ Core skills (SQL, Python, ETL, data modeling) — weeks 1-5
- ✅ Cloud fundamentals (AWS) — weeks 6-7
- ✅ Modern skills (APIs, LLM integration) — week 9
- ✅ Interview readiness starts EARLY — week 10
- ✅ Heavy interview practice — weeks 10-14
- ✅ 4 portfolio projects with SG/MY context
